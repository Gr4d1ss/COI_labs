# Лабораторная работа 5. Цветовые модели. Цветовая сегментация

## Цель работы
Научиться применять методы конвертации цветовых моделей и выполнить цветовую сегментацию изображения.

## Теоретические основы

### Цветовая модель RGB
- **Red, Green, Blue** — три основных цвета
- Система координат: куб от ⟨0,0,0⟩ (чёрный) до ⟨255,255,255⟩ (белый)
- Недостаток: не разделены яркостная и оттеночная компоненты

### Цветовая модель HSV
- **Hue (H)** — цветовой тон (0-180° в OpenCV)
- **Saturation (S)** — насыщенность (0-255)
- **Value (V)** — яркость (0-255)
- Преимущество: соответствует восприятию цвета человеком

### Цветовая модель LAB
- **L** — яркость (Lightness)
- **A** — компонент от зелёного к красному
- **B** — компонент от синего к жёлтому
- Преимущество: независимое представление яркости и цвета

## Как работает код

### 1. Загрузка и конвертация изображения
```python
image = cv2.imread('mandarin.jpg')
image_rgb = cv2.cvtColor(image, cv2.COLOR_BGR2RGB)
image_hsv = cv2.cvtColor(image_rgb, cv2.COLOR_RGB2HSV)
image_lab = cv2.cvtColor(image_rgb, cv2.COLOR_RGB2LAB)
```
- Исходное изображение в BGR (формат OpenCV)
- Конвертируем в RGB для matplotlib
- Преобразуем в HSV и LAB для сегментации

### 2. Цветовая сегментация в HSV
```python
lower_orange_hsv = np.array([5, 50, 50])
upper_orange_hsv = np.array([20, 255, 255])
mask_hsv = cv2.inRange(image_hsv, lower_orange_hsv, upper_orange_hsv)
```
**Как работает:**
- `H ∈ [5, 20]` — диапазон оранжевого оттенка (в OpenCV H делится на 2, реальный диапазон 10-40°)
- `S ∈ [50, 255]` — достаточная насыщенность (отсекаем серые пиксели)
- `V ∈ [50, 255]` — достаточная яркость (отсекаем тёмные пиксели)
- `cv2.inRange()` создаёт бинарную маску: 255 где пиксель в диапазоне, 0 иначе

### 3. Цветовая сегментация в LAB
```python
lower_orange_lab = np.array([0, 100, 100])
upper_orange_lab = np.array([255, 255, 255])
mask_lab = cv2.inRange(image_lab, lower_orange_lab, upper_orange_lab)
```
**Как работает:**
- `L ∈ [0, 255]` — любая яркость
- `A ∈ [100, 255]` — сильные красные оттенки (мандарины красновато-оранжевые)
- `B ∈ [100, 255]` — сильные жёлтые оттенки
- Комбинация A и B даёт оранжевый цвет

### 4. Применение маски
```python
result_hsv = cv2.bitwise_and(image_rgb, image_rgb, mask=mask_hsv)
```
**Как работает:**
- `cv2.bitwise_and()` применяет маску к изображению
- Пиксели где маска=255 сохраняются, где маска=0 становятся чёрными

### 5. Визуализация результатов
```python
plt.figure(figsize=(15, 5))
plt.subplot(1, 4, 1)
plt.imshow(image_rgb)
plt.title("Original Image")
plt.axis('off')
```
- `plt.subplot(1, 4, 1)` — создаёт сетку 1×4, выбирает первую ячейку
- `plt.imshow()` — отображает изображение
- `plt.savefig()` — сохраняет результат в файл

### 6. Построение гистограмм
```python
h_channel = image_hsv[:, :, 0]
plt.hist(h_channel.flatten(), bins=180, range=[0, 180])
```
**Как работает:**
- `h_channel.flatten()` — превращает 2D массив в 1D
- `bins=180` — 180 корзинок для гистограммы (по количеству значений H)
- `plt.axvline()` — рисует вертикальные линии порогов

## Ключевые выводы
1. **HSV лучше для сегментации по цвету** — Hue напрямую кодирует оттенок
2. **LAB лучше для сложных случаев** — разделение яркости и цвета более точное
3. **Пороги подбираются экспериментально** — через анализ гистограмм
4. **Маска — бинарное изображение** — белый = объект, чёрный = фон
